# Project Overview
- Purpose: Next.js 기반의 미니멀 픽셀 아트 스타일 개인 블로그.
- Core features: MDX 포스팅, 카테고리/페이지네이션, 다크/라이트 테마, SEO(robots/sitemap), Giscus 댓글.
- Tech stack: Next.js App Router (v16), React 19, TypeScript, Tailwind CSS v4, MDX 파이프라인(next-mdx-remote + remark/rehype), Lucide icons.
- Deployment target: 정적 내보내기 기반 배포(Vercel/GitHub Pages).
- Main data source: `_posts` 디렉토리의 `.mdx` 포스트 + frontmatter.
