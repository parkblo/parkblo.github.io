# Suggested Commands
- Install deps: `npm install` 또는 `npm ci`
- Dev server: `npm run dev`
- Production build: `npm run build`
- Start production server: `npm run start`
- Lint: `npm run lint`
- Fast code search: `rg "pattern" src`
- Git status: `git status --short`
- Diff check: `git diff -- <path>`
